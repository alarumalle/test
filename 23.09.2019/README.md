#Päeva teemad

## Väljund
```
<?php
print "Tekst mida soovid väljastada";
?>
```
või
```
<?php
echo "Tekst mida soovid väljastada";
?>
```
## Muutujad

```
$muutujaNimi = "väärtus";
```

```
<?php
echo "Mingi text".$muutujaNimi." ja veel mingi tekst";
?>
```
